# 👩‍💻 Margaret Loraine T. Malaluan | Student Portfolio

Welcome to my professional portfolio! This portfolio showcases my journey as a **Diploma in Information Technology** student at the **Polytechnic University of the Philippines**, my basic projects, and my involvement in various activities within the inside and  outside of the university .

## 🚀 Live Demo
You can view the live site here: [margamalaluan20.github.io](https://margamalaluan20.github.io/)

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Bento Grid Design)
* **Framework:** Bootstrap 5.3
* **Icons:** Bootstrap Icons
* **Interactive Elements:** JavaScript (ES6+), Lightbox for image viewing
* **Deployment:** GitHub Pages

## 🌟 Key Features
* **Bento Grid Layout:** A modern, responsive dashboard-style layout to showcase skills and experience.
* **Interactive Awards Toggle:** A custom-built awards section with a dynamic toggle button and "click-outside-to-close" functionality.
* **Project Gallery:** High-quality previews of technical projects including AI-SAM, Data-Wise Kadakamp, and Network Analysis.
* **Dark/Light Mode:** Seamless theme switching for better user experience.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
